Graphic Design/

UX Design

COMN is an application that lets users share their clothes with their friends so they can try new items and experiment with different styles, while doing the planet and your wallet a favor.

For COMN I created their logo and style guide. To improve the design of the application I organize consumer testing sessions to uncover the design improvements in the application. The user testing sessions are done using interview techniques and prototypes I created in both Adobe XD and Figma. Furthermore, together with the developer of the application I conduct A/B testing. The goal of these sessions is to create the most intuitive experience for the users of the COMN application.
